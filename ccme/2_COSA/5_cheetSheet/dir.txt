


# 1. USER & AUTHENTICATION FILES

| File/Dir                    | Purpose                | Possible Trap             |
| --------------------------- | ---------------------- | ------------------------- |
| `/etc/passwd`               | User account info      | Password NOT stored here  |
| `/etc/shadow`               | Encrypted passwords    | Root-only readable        |
| `/etc/group`                | Group info             | Group passwords may exist |
| `/etc/gshadow`              | Secure group passwords | Similar to shadow         |
| `/home/username`            | User home directory    | Not config file           |
| `/root`                     | Root user home         | NOT `/home/root`          |
| `/etc/login.defs`           | User defaults          | UID/GID ranges            |
| `/etc/skel`                 | Default home template  | Copied during useradd     |
| `/etc/sudoers`              | sudo permissions       | Edit using `visudo`       |
| `/etc/security/limits.conf` | User resource limits   | ulimit related            |

# 2. BOOTING / GRUB / SYSTEMD

| File/Dir                   | Purpose                | Trap                       |
| -------------------------- | ---------------------- | -------------------------- |
| `/boot/grub2/grub.cfg`     | GRUB config            | Auto-generated             |
| `/etc/default/grub`        | Editable GRUB defaults | NOT actual boot menu       |
| `/boot/vmlinuz-*`          | Linux kernel image     | Kernel binary              |
| `/boot/initramfs-*`        | Initial RAM FS         | Temporary root FS          |
| `/etc/fstab`               | Permanent mounts       | Wrong entry may break boot |
| `/etc/systemd/system/`     | Custom unit files      | Higher priority            |
| `/usr/lib/systemd/system/` | Default unit files     | Vendor supplied            |
| `/etc/inittab`             | Old SysV init config   | Mostly obsolete in systemd |

# 3. FILESYSTEM / STORAGE / LVM

| File/Dir       | Purpose             | Trap                          |
| -------------- | ------------------- | ----------------------------- |
| `/etc/fstab`   | Auto-mount config   | `mount -a` tests it           |
| `/etc/mtab`    | Mounted filesystems | Often symlink                 |
| `/proc/mounts` | Kernel mount info   | Runtime generated             |
| `/dev/sda`     | Entire disk         | Not partition                 |
| `/dev/sda1`    | Partition           | Number matters                |
| `/etc/lvm/`    | LVM configs         | VG/LV metadata                |
| `/swapfile`    | Swap file           | Alternative to swap partition |

# 15. SPECIAL DIRECTORIES

| Directory       | Purpose                       | Trap                    |
| --------------- | ----------------------------- | ----------------------- |
| `/proc`         | Virtual kernel/process FS     | Not real disk files     |
| `/sys`          | Kernel device interface       | sysfs                   |
| `/dev`          | Device files                  | Hardware abstraction    |
| `/tmp`          | Temporary files               | Often cleaned at reboot |
| `/var`          | Variable/changing data        | Logs/spool/cache        |
| `/opt`          | Optional third-party software | Extra packages          |
| `/bin`          | Essential binaries            | User commands           |
| `/sbin`         | System admin binaries         | Root commands           |
| `/usr/bin`      | User programs                 | Most commands           |
| `/usr/sbin`     | Admin commands                | System tools            |
| `/lib` `/lib64` | Shared libraries              | Dynamic linking         |

# VERY COMMON CDAC TRAPS

| Trap                               | Correct                             |
| ---------------------------------- | ----------------------------------- |
| `/root` vs `/home/root`            | `/root`                             |
| `sshd_config` vs `ssh_config`      | server vs client                    |
| `main.cf` vs `master.cf`           | Postfix configs                     |
| `authorized_keys` vs `known_hosts` | allowed keys vs remote fingerprints |
| `/etc/hosts` vs DNS                | local mapping overrides DNS         |
| `fstab` vs `mtab`                  | static vs current mounts            |
| `grub.cfg` vs `/etc/default/grub`  | generated vs editable               |
| `systemctl start` vs `enable`      | now vs boot                         |
| `reload` vs `restart`              | config reload vs full restart       |
| `firewalld` vs `iptables`          | modern vs older                     |
| `chronyd` vs `ntpd`                | newer vs older NTP                  |
| `access_log` vs `error_log`        | requests vs errors                  |
| `A` vs `PTR`                       | hostnameâ†’IP vs IPâ†’hostname          |
| `IMAP` vs `POP3`                   | server mail vs download             |
| `Bridge` vs `NAT`                  | direct LAN vs hidden VM             |
| `Hard link` vs `Soft link`         | inode shared vs pointer             |
| `root_squash` vs `no_root_squash`  | security vs root preserved          |
