
# IEEE Standards Quick Revision

| Standard | Technology | Frequency / Purpose | Important Point                          |
| -------- | ---------- | ------------------- | ---------------------------------------- |
| 802.3    | Ethernet   | Wired LAN           | Traditional Ethernet standard            |
| 802.11   | Wi-Fi      | Wireless LAN        | Base Wi-Fi family                        |
| 802.11a  | Wi-Fi      | 5 GHz               | Faster, less interference, shorter range |
| 802.11b  | Wi-Fi      | 2.4 GHz             | Older, slower, longer range              |
| 802.11g  | Wi-Fi      | 2.4 GHz             | Faster than b                            |
| 802.11n  | Wi-Fi      | 2.4 + 5 GHz         | Better speed and MIMO                    |
| 802.11ac | Wi-Fi 5    | Mainly 5 GHz        | High speed modern Wi-Fi                  |
| 802.11ax | Wi-Fi 6    | 2.4 + 5 GHz         | Better efficiency and dense environments |
| 802.1Q   | VLAN       | VLAN Tagging        | Used in trunk links                      |
| 802.5    | Token Ring | Ring topology       | IBM old technology                       |

# Wireless Standards Memory Trick

| Standard | Speed  | Range   | Interference      |
| -------- | ------ | ------- | ----------------- |
| 2.4 GHz  | Lower  | Longer  | More interference |
| 5 GHz    | Higher | Shorter | Less interference |

So:
2.4 GHz = range
5 GHz = speed

# OSI Layers Ultra Fast Revision

| Layer          | Main Work           | Keywords                             |
| -------------- | ------------------- | ------------------------------------ |
| 7 Application  | User services       | HTTP, FTP, SMTP                      |
| 6 Presentation | Formatting          | Encryption, Compression, Translation |
| 5 Session      | Session management  | Synchronization, Dialogue            |
| 4 Transport    | Reliable delivery   | TCP, UDP, Ports, Retransmission      |
| 3 Network      | Routing             | IP, Router, Path selection           |
| 2 Data Link    | Local delivery      | MAC, Frame, CRC, Switching           |
| 1 Physical     | Signal transmission | Bits, Cables, Signals                |

---

# Layer Wise Important Characteristics

| Layer        | Characteristics                                                              |
| ------------ | ---------------------------------------------------------------------------- |
| Application  | Closest to user, provides network services                                   |
| Presentation | Compression, encryption, syntax translation                                  |
| Session      | Session establishment and synchronization                                    |
| Transport    | Ordered delivery, error checking, retransmission, segmentation, flow control |
| Network      | Logical addressing and routing                                               |
| Data Link    | Framing, MAC addressing, CRC error detection                                 |
| Physical     | Converts bits into electrical/light/radio signals                            |

# TCP vs UDP

| Feature            | TCP                 | UDP               |
| ------------------ | ------------------- | ----------------- |
| Reliability        | Yes                 | No                |
| ACK                | Yes                 | No                |
| Speed              | Slower              | Faster            |
| Connection         | Connection-oriented | Connectionless    |
| Sequencing         | Yes                 | No                |
| Retransmission     | Yes                 | No                |
| Streaming Suitable | Less                | Best              |
| Examples           | HTTP, HTTPS, FTP    | Streaming, Gaming |

# Extremely Important Exam Confusions

| Confusion                 | Correct Logic                                            |
| ------------------------- | -------------------------------------------------------- |
| MAC vs IP                 | MAC local, IP end-to-end                                 |
| Collision vs Interference | Wireless mostly interference                             |
| Hub vs Switch             | Hub broadcasts, switch learns MAC                        |
| TCP vs UDP                | Reliability vs speed                                     |
| STP vs Fiber              | STP resists EMI, Fiber immune to EMI                     |
| Session vs Presentation   | Session manages communication, Presentation formats data |
